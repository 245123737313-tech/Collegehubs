PROJECT: College Hub - Student Portal
TECH STACK: HTML, CSS, JavaScript, jQuery, AngularJS, PHP, MySQL
SERVER: XAMPP (Apache + MySQL)

================================================================================
                         INSTALLATION (5 MINUTES)
================================================================================

1. START XAMPP
   - Open XAMPP Control Panel
   - Click "Start" for Apache
   - Click "Start" for MySQL
   - Wait for both to show GREEN

2. CREATE DATABASE
   - Open: http://localhost/phpmyadmin
   - Click "New" ‚Üí Create database "collegehub_db"
   - Go to "SQL" tab
   - Copy content from: database/schema.sql
   - Paste and click "Go"

3. COPY PROJECT FILES
   - Copy "collegelibraryhub" folder
   - Paste in: C:\xampp\htdocs\

4. OPEN APPLICATION
   - Go to: http://localhost/collegelibraryhub/
   - You should see the College Hub home page!

================================================================================
                            FIRST STEPS
================================================================================

REGISTER:
  1. Click "Register" button (top right)
  2. Fill in: Name, Email, Phone, Password
  3. Click "Register"

LOGIN:
  1. Click "Login" button
  2. Enter your email and password
  3. Click "Login"

POST A NOTICE:
  1. Click "Notices" in navigation
  2. Click "Add Notice"
  3. Enter title and description
  4. Click "Post Notice"

POST A LOST ITEM:
  1. Click "Lost & Found" in navigation
  2. Click "Post Lost Item"
  3. Enter item details
  4. Click "Post Item"

CALCULATE CGPA:
  1. Click "CGPA Calculator" in navigation
  2. Enter GPA and credits
  3. Click "Add Semester" for more
  4. CGPA updates automatically

================================================================================
                          TROUBLESHOOTING
================================================================================

DATABASE ERROR?
  ‚Üí Make sure MySQL is running (green in XAMPP)
  ‚Üí Check if database "collegehub_db" exists in phpMyAdmin

PAGE NOT LOADING?
  ‚Üí Make sure Apache is running (green in XAMPP)
  ‚Üí Check if folder is in C:\xampp\htdocs\collegelibraryhub\
  ‚Üí Clear browser cache (Ctrl+Shift+Delete)

FORMS NOT WORKING?
  ‚Üí Press F12 to open browser console
  ‚Üí Check for error messages
  ‚Üí Make sure PHP files are in backend/ folder

================================================================================
                          DATABASE TABLES
================================================================================

users
  - id (INT, Primary Key)
  - name (VARCHAR 100)
  - email (VARCHAR 100, Unique)
  - phone (VARCHAR 15)
  - password (VARCHAR 255)
  - created_at (TIMESTAMP)

notices
  - id (INT, Primary Key)
  - title (VARCHAR 200)
  - description (TEXT)
  - date (TIMESTAMP)

lostfound
  - id (INT, Primary Key)
  - item_name (VARCHAR 150)
  - description (TEXT)
  - posted_by (VARCHAR 100)
  - date (TIMESTAMP)

================================================================================
                          API ENDPOINTS
================================================================================

Authentication:
  POST   /backend/register.php        - Register new user
  POST   /backend/login.php           - Login user
  GET    /backend/logout.php          - Logout user
  GET    /backend/check_session.php   - Check login status

Notices:
  GET    /backend/fetch_notices.php   - Get all notices
  POST   /backend/add_notice.php      - Add new notice

Lost & Found:
  GET    /backend/fetch_lostitems.php - Get all lost items
  POST   /backend/add_lostitem.php    - Add new lost item
